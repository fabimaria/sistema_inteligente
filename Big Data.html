Big Data

Refere a uma base de dados com uma quantidade enorme de dados, podendo estar alocada em múltiplos servidores, e estes dados não precisam estar estruturados necessariamente, como ocorrem nos bancos de dados relacional, ou seja, com as organizações lidam um enorme crescimento de dados, sendo necessário pensar em formas de analisar e processar esses dados, porém o Big Data, vem sendo utilizado para definir o fluxo de dados desetruturados cuja a estrutura de dados relacionais é incapaz de processar. São dados muito grandes ou muito complexos para serem gerenciados por técnicas tradicionais de processamento, análise e armazenamento dos dados, a ciência que analisa dados brutos para extrair conhecimento útil deles.
